## Alicia In Theory
Alicia is a simple programming language with a focus on being easy to pick up and learn, and easy to integrate into an existing project to extend with scripting.

You can read the Alicia book [here](https://www.google.com) to learn more about the language.

## Alicia In Practice
You can check out a simple mine-sweeper game implementation with Alicia [here](https://www.google.com) to get a feel for how the language
can be put to use in a practical way, or find example code in the Alicia book.
